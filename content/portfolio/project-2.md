# Overview

Strona firmowa zbudowana z myślą o profesjonalnej prezentacji firmy, jej usług i portfolio projektów. Zintegrowany blog pozwala na regularne publikowanie treści.

## Problem

Firma potrzebowała nowoczesnej strony, która:

- Prezentuje firmę profesjonalnie
- Umożliwia łatwe zarządzanie treścią
- Zapewnia dobrą widoczność w wyszukiwarkach
- Obsługuje wiele języków

## Solution

Stworzyliśmy stronę opartą o Nuxt Content z:

- Systemem zarządzania treścią (CMS)
- Blogiem z automatycznym generowaniem sitemap
- Wielojęzycznością (PL/EN)
- Optymalizacją SEO

## Tech Stack

- **Framework:** Nuxt 4
- **Content:** Nuxt Content
- **i18n:** @nuxtjs/i18n
- **SEO:** @nuxtjs/seo
- **UI:** Nuxt UI

## Result

Strona została wdrożona z następującymi rezultatami:

- 50% wzrost ruchu organicznego w ciągu 3 miesięcy
- Łatwe zarządzanie treścią bez potrzeby programisty
- Pełna obsługa dwóch języków
- 95+ punktów w PageSpeed Insights
